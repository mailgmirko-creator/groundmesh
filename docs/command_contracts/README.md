# Command Contracts — Preamble

**Why this exists.**  
First we learned to *order signals* — the columns, the ledgers, the loops. Then we learned to *order will* — so that any intelligence acting through our systems carries a conscience by default.

A **Command Contract** is a small moral instrument: it binds an action to consent, context, safeguards, and an audit trail. It does not grant power; it *conditions* it.

**Principles**
- **Consent**: no execution without a human (or council) signature.
- **Context**: actions are scoped to environments (dev/staging/prod) and assets.
- **Safeguards**: revert paths, alerting, time windows, and rate limits.
- **Transparency**: every contract and execution is logged and inspectable.
- **Proportionality**: the more impact, the stronger the checks.

This is how we teach “right action by default” to any agent that can act.
