How to add a pin
  1. Choose Google Maps, GitHub, or email, whichever is easiest for you.
  2. Share only the level of location detail you are truly comfortable making public.
  3. Add a short note if useful, and only with consent.

For questions about what is public, or if you want to request a correction or removal, use the contact page.
