Registration status

GroundMesh registration is not broadly open yet.

GroundMesh intends to support real declared joining and wider participation in time, but that path must begin as a careful pilot with real moderation, privacy, consent, and rollback gates behind it.

For now, use the current public paths below. Do not treat this site as a confidential intake or emergency registration system.

Open Contributors Hub Contact GroundMesh Privacy Atlas

What works right now

  • First contact through the public contact page
  • Low-friction participation through the contributors hub
  • Public map pin requests with explicit consent
  • Builder participation through Get Started

What is not open yet

  • Broad public registration at planetary scale
  • Confidential or sensitive identity intake on the static docs site
  • Emergency handling promises beyond the current public trust layer
  • Automated acceptance without human review

How it will open

The first real registration path should open as a narrow pilot, not a symbolic mass launch. Scope, consent, moderation, operations, and rollback must all be honestly green first.
