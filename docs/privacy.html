GroundMesh Privacy

This page states the current privacy posture of the public GroundMesh layer as plainly as possible.

Current public posture

  • The static docs pages do not themselves provide a secure private intake system.
  • Public pages should collect as little personal data as possible.
  • If you submit a public map pin, only share what you truly consent to make public.
  • Email and GitHub are external systems with their own privacy characteristics.

What this means in practice

Do not treat the current public site as a confidential channel. For now it is a public seed, a coordination doorway, and a trust-building layer. It is not yet a hardened platform for sensitive disclosures or emergency reporting.

Data minimization rule

GroundMesh should ask only for the smallest amount of information needed for the current purpose. If a simpler or less revealing path works, prefer that path.
